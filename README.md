<h1>How to Start Project</h1>

<h3>Frontend</h3>
<ul>
  <li>
    Go to frontend
  </li>
  <li>
    run command "npm install"
  </li>
  <li>
    run command "npm start"
  </li>
</ul>

Now open another terminal

<h3>Backend</h3>
<ul>
  <li>
    Go to backend
  </li>
  <li>
    create a .env file and write the content
  </li>
  <li>
    run command "npm install"
  </li>
  <li>
    run command "npm run dev"
  </li>
</ul>
